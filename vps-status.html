System Information

  • Status checks are performed automatically every 30 seconds
  • Click "Refresh" to manually check status
  • VPS status checks server response time and availability
  • Database status requires PHP backend API to be configured
  • Current mode: Static HTML site (no database backend)
  • To enable database monitoring, add PHP API endpoint at /api/status.php
  • PostgreSQL support: Make sure PHP pgsql extension is installed
  • Configure PostgreSQL credentials in api/status.php
